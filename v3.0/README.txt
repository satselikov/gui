ADC.py - ADC methods for voltage and current + helper methods for calculations
EVB.py - submit and calibrate functions used by the power supply GUI
LED.py - special functions for the LED, LED_sub (submit)
PowerSupply.py - Each component on the power supply GUI creates an instance of this class. Getters and Setters for every variable included.
clock.py - clock click and submit functions 
extras.py - extra functions, methods called a couple of times 
main.py - main driver, contains all the GUI attributes (buttons, entries, frames, tabs, mainloop) for both the Power Supply and ASIC GUIs.
